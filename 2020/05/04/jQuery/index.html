与原生js的区别

  • 原生js会等到DOM元素加载完成，并且图片也加载完毕才会执行
  • jquery会等到dom元素加载完毕，但不会等到图片也加载完毕就执行
  • 多个入口函数写入时，原生js会最后一个window.onload会覆盖之前的入口函数
  • jquery不会覆盖之前的入口函数

map与each的区别

  • 返回值

    • map返回的是一个空数组
    • each返回的是一个数组，这个数组是遍历的数组本身

  • return

    • each静态方法不支持在回调函数中对遍历的数组进行处理
    • map静态方法可以在回调函数中通过return对遍历的数组进行处理，然后生成一个新的数组
