字体样式

  • 字体类型：font-family
  • 字体大小：font-size
  • 字体粗细：font-weight
  • 字体风格：font-style
  • 字体颜色：color

文本样式

  • 首行缩进：text-indent
  • 水平对齐：text-align
  • 文本修饰：text-decoration(上划线、下划线、删除线)
  • 大小写：text-transform
  • 行高：line-height
  • 字母间距、词间距：letter-spacing、word-spacing

边框样式

  • 边框的宽度：border-width
  • 边框的外观：border-style(虚线：dashed;实线 solid)
  • 边框的颜色：border-color

局部样式：

  • border-top(上边框)
  • border-buttom(下边框)
  • border-left(左边框)
  • border-right(右边框)

超链接样式

超链接伪类

伪类 说明
a:link 定义a元素为访问时的样式
a:visited 定义a元素访问后的样式
a:hover 定义鼠标经过a元素时的样式
a:active 定义鼠标点击激活时的样式

注意：必须按照顺序进行，不然浏览器可能无法正常显示

扩展：hover不仅适用与超链接标签，而且任何标签都可以适用（它是个好东西）

盒子模式

weight、height

两个属性可以指定块元素的宽度和高度，在没有宽度的设置时，自动默认占满整行。（行内元素无法设置这两个值，因此行内元素的高度和宽度由内容撑起）

padding

指内容区到边框的距离

margin

指定两个元素之间的距离，如果同时存在父元素和兄弟元素，先以兄弟元素判断。没有兄弟元素才与父元素对比。

border

指定元素的边框大小、实线或者虚线、颜色。
